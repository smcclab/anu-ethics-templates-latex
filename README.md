# anu-ethics-templates-latex

ANU Ethics Templates in LaTeX for computery people, can be imported directly to Overleaf

This repository includes templates for:

- Project Description: `project_description.tex`
- Participant Information Sheet (PIS): `participant_information_sheet.tex`
- Consent Form: `consent_form.tex`
- Recruitment Resources: `recruitment_resources.tex`
- Sample Interview Questions / Questionnaire: `sample_questions.tex`

A brief explanation and instructions are provided in `main.tex`.

This repository can be downloaded as a `.zip` file and uploaded to Overleaf to continue work there.

Please note, you will need to change the LaTeX compiler to `XeLaTeX` in Overleaf's settings for your project.

A `Makefile` is provided to build all files in the project.
