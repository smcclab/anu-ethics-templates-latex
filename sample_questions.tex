\section{Interview Questions}

The interview will be conducted in a semi-structured manner from the following questions:

\begin{itemize}
    \item What were your first impressions of the prototype system?
    \item How did you learn to use the prototype within the session?
    \item What do you feel you achieved with the prototype?
    \item What did you like most about the system?
    \item What did you like least about the system?
    \item Do you have any suggestions for improving the system?
    \item Is there anything else you would like to share about your experience?
\end{itemize}

\section{Survey Questions}

We will administer the the System Usability Scale (SUS) which is a standard written questionnaire of usability. There are ten questions with 5-point Likert responses from "Strongly Disagree" to "Strongly Agree". The questions are as follows:

\begin{enumerate}
\item I think that I would like to use this system frequently.
\item I found the system unnecessarily complex.
\item I thought the system was easy to use.
\item I think that I would need the support of a technical person to be able to use this system.
\item I found the various functions in this system were well integrated.
\item I thought there was too much inconsistency in this system.
\item I would imagine that most people would learn to use this system very quickly.
\item I found the system very cumbersome to use.
\item I felt very confident using the system.
\item I needed to learn a lot of things before I could get going with this system.
\end{enumerate}
